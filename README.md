# OpenBit Downloads Repository

This repository contains torrent files used by the main OpenBit website.

Main website repository:

- `https://github.com/your-username/openbit`

## Structure

```text
openbit-downloads/
|- games/
`- software/
```

## Usage

1. Add `.torrent` files to:
   - `games/`
   - `software/`
2. Keep file names clean and descriptive.
3. Do not add unrelated files.

## Integration with OpenBit Website

The website expects these folders under its `downloads/` directory:

- `downloads/games/`
- `downloads/software/`

You can sync this repository into the website project as `downloads/`.

## Notes

- Use only files you have rights to distribute.
- Keep categories consistent (`games` vs `software`).
